Understanding Scanning Types

Dynamic Analysis

Dynamic analysis involves executing the file in a secure, isolated environment to observe its real-time behavior. This includes monitoring system calls, network activity, and registry changes. It helps detect advanced threats that evade signature-based detection, Based on the changes that malware made on the enivornement our trained machine learning model will take the decision if the file is a malware or not. Please keep in mind if you chose this option the scan may take up to 6 minutes.

Static Analysis

Static analysis examines the file without execution by analyzing its code, structure, and metadata. This method compares the file against known malware signatures and heuristics. We use VirusTotal API to gather results from multiple antivirus engines for better accuracy.
